# Credit Score System 🚀

A backend REST API project built using Spring Boot, MySQL, JPA, and Hibernate.

This project demonstrates complete CRUD operations for managing users and their credit scores.

---

## 🛠 Technologies Used

- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- Postman
- Git & GitHub

---

## 📌 Features

✅ Create User  
✅ Get All Users  
✅ Update User  
✅ Delete User  
✅ REST APIs using Spring Boot  
✅ MySQL Database Integration  
✅ Layered Architecture

---

## 📂 Project Structure

```txt
controller  → REST APIs
service     → Business Logic
repository  → Database Operations
entity      → Database Table Mapping


## 🚀 API Endpoints

### Create User

POST `/users`

### Get All Users

GET `/users`

### Update User

PUT `/users/{id}`

### Delete User

DELETE `/users/{id}`
